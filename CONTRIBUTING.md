# Contributing to Whisper Key

Thanks for your interest in contributing!

## Before You Start

**Please open an issue before submitting feature PRs.** This lets us discuss whether the feature fits the project's direction before you invest time building it.

Check the [roadmap](docs/roadmap/roadmap.md) to see what's planned and what's available to work on.

## Guidelines

- Bug fixes and small improvements are welcome as direct PRs
- New features need an issue discussion first
- Follow existing code conventions and project structure
- This project intentionally avoids GUI frameworks — keep it lightweight

## Questions?

Join the [Discord](https://discord.gg/uZnXV8snhz) to chat about ideas before diving in.
